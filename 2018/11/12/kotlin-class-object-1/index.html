编译期常量

可以使用const关键字将常量标记为编译期常量，这些属性需要满足以下条件：

  • 位于顶层或者是object声明或companion object的一个成员
  • 以String或原生类型值初始化
  • 没有自定义getter

可见性修饰符

在Kotlin中有四种可见修饰符：

  • private
    只会在声明它的文件内可见
  • protected
    不适用于顶层声明
  • internal
    在相同模块内随处可见
  • public
    声明将随处可见

数据类

主构造函数声明的属性

  • equals()/hashCode()对
  • toString()
  • componentN()
  • copy()

数据类需要满足的条件

  • 主构造函数需要至少一个参数
  • 主构造函数的所有参数需要标记为val或var
  • 数据类不能是抽象、开放、密封或者内部

密闭类

密闭类用来表示受限的类继承结构：当一个值为有限集中的类型、而不能有任何其他类型时。密闭类可以有子类，但是所有子类都必须在密闭类自身相同的文件中声明。密闭类自身是抽象的，它不能直接实例化并可以有抽象成员。密闭类不允许有非private构造函数。而扩展密闭类子类的类可以放在任何位置，并非需要在同一个。
